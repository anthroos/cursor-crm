# Contributing

Thanks for your interest in contributing! This project is part of the [Claude Code tools suite](https://github.com/anthroos).

## How to Contribute

1. **Fork** the repository
2. **Create a branch** for your change (`git checkout -b feature/my-change`)
3. **Make your changes** and test them
4. **Commit** with a clear message
5. **Open a Pull Request** with a description of what you changed and why

## What We're Looking For

- Bug fixes
- New validation checks
- Documentation improvements
- Integration guides for new channels
- Schema improvements
- Sample data improvements

## Code Style

- Python: follow PEP 8
- Use descriptive variable names
- Keep functions focused and small
- Add comments only where the logic isn't self-evident

## Reporting Issues

Open a [GitHub Issue](../../issues) with:
- What you expected to happen
- What actually happened
- Steps to reproduce

## Questions?

Open an issue or start a discussion. We're happy to help.
